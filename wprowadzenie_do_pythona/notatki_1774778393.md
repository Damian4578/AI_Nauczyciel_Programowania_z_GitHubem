# Wprowadzenie do Pythona

## 1. Historia języka Python

- Język stworzony przez Guido van Rossumiem (1991)
- Nazwa języka pochodzi z imiona muzyka Davida Pythona 
- Wersja 3.10 (obecnie najnowsza)

## 2. Podstawy języka Python

- Deklaracja zmiennej: `x = 5`
- Typy danych:
  - Liczby całkowite: `int`
  - Liczby zmiennoprzecinkowe: `float`
  - Tekst: `str`
  - Logika: `True`, `False`
  - None
- Operatorzy:
  - `+` dodawanie
  - `-` odejmowanie
  - `*` mnożenie
  - `/` dzielenie

## 3. Zmienne i typy danych

```python
x = 5  # zmienna liczbowej
y = 3.14  # zmienna zmiennoprzecinkowa
imie = "Jan"  # zmienna tekstu
wiek = True  # zmienna logiczna
```

## 4. Funkcje w Pythonie

- Funkcja: `def suma(a, b): return a + b`
- Wywołanie funkcji: `suma(5, 5)`
- Argumenty: `a` i `b` są argumentami, których używa funkcja

## 5. Podstawy kontroli programu

- If else: `if x > 0: return "X sądu większe" else: return "X sądu mniejsze" `
- While loop: `i = 0` `while i < 10: print(i)` `i += 1`
- For loop: `słownik = {1: 'a', 2: 'b'}` `for key, value in słownik.items(): print(key, value)`

```Python
# Podstawowy przykład While loop
i = 0 
while i < 10: 
    print(i)
    i += 1
```

## 6. Listy w Pythonie

- Lista: `lista = [1, 2, 3, "a", "b", 3.14, True]`
- Indeksowanie: `lista[0]`
- Wysłanie w przestrzeni: `print(*lista)`
- Zastosowanie funkcji: `print(sum(lista))`

## 7. Zbiory w Pythonie

- Zbiór: `zestaw = {1, 2, 2, 3, 4, 5}`
- Dodawanie do zbioru: `zestaw.add(6)`
- Usuwanie z zbioru: `zestaw.remove(2)`
- Sprawdzanie elementów w zestawie: `1 in zestaw`
- Sprawdzanie nie przynależenia do zestawu: `4 not in zestaw`

## 8. Słowniki w Pythonie

- Słownik: `slownik = {1: 'a', 2: 'b', 3: 'c', 4: 'd'}`
- Dodawanie do słownika: `slownik[5] = 'e'`
- Usuwanie z słownika: `del slownik[5]`
- Sprawdzanie klucza: `3 in slownik`
- Sprawdzanie elementu: `'c' in slownik.values()`

## 9. Listy i operacje na nich

- Podstawy:
  - indeksowanie: 1, 2, 3
  - dodawanie elementów: `[1, 2, 3] + [1, 1]`
  - kopiowanie: `zamiejscowienie = lista[::]`
  - powtarzanie: `[1, 1, 1, 1]`
- Indeksowanie od końca: `lista[-1]`
- Sprawdzanie indeksowania: `1 in lista`

## 10. Wywoławanie funkcji i wyświetlanie komunikatów

- Wywołanie funkcji: `print("Wieme, wieme")`
- Wypisanie komunikatu: `wyswietlenie = "Wyświetlenie"`

```Python
# Przykład wywoławania funkcji i wypisanie komunikatu
def funkcja_a():
    print("Wywoławanie funkcji")
def funkcja_b():
    return "Zwracanie"
print(funkcja_b())
funkcja_a()
```

## 11. Klasa w Pythonie

- Klasa: `class Przyklad: pass`
- Konstruktor: `def __init__(self): pass`
- Atrybuty: `klasa.Przyklad.__dict__`
- Metody: `klasa.Przyklad`
- Obiekt: `pierwszy_przy_klase = Przyklad()`

```python
# Przykład klasy Pythona
class Pierwszy:
    def __init__(self):
        self.atr1 = "Atrybut 1"
        self.__atrubut2 = "Nieoznaczona"
    def metodka(self):
        return "Metoda"
    def metodka(self):
        return "Dodatek"
pierwszy = Pierwszy()
print(pierwszy.atr1)
print(pierwszy.metodka())
print(pierwszy.__atrubut2)
```

## 12. Aby skończyć, oto przykład wypisania tekstu do pliku

- Otwieranie pliku: `plik = open('plik.txt', 'w')`
- Zapisywanie tekstu: `plik.write('To jest tekst wejsciowy')`
- Zapisowanie tekstowego oznakowania wejcia: `plik.write("\n")`
- Zamknięcie pliku: `plik.close()`

```python
# Przykład wypisania do pliku
plik = open('plik.txt', 'w')
plik.write('To jest tekst wejściowy \n')
plik.write('Kolejny wejście')
plik.close()
```
