### Wprowadzenie do Pyt

### Instalacja środowiska programu

Aby rozpocząć naukę języka `Pyt`, należy zainstalować środowisko programu. W tym celu należy:

#### Instalacja z użyciem pip
```bash
pip install pyt
```

#### Instalacja z użyciem conda
```bash
conda install pyt
```
### Podstawy języka

- Typy danych
- Funkcje
- Pętle
- Funkcje analityczne, np. `max()`, `min()`, `sum()`

### Podstawy funkcji

- Funkcja rekurencywna
- Funkcja z argumentami
- Funkcja z powrotami

### Podstawy pętli

- Pętla for
- Pętla while
- Pętla do

### Podstawy typów danych

- Liczby całkowite (int)
- Liczby dziesiętne (float)
- Ciągi znaków (str)

### Biblioteki Python

- `math`
- `random`
- `time`

### Przykład użycia biblioteki `math`
```python
import math

print(math.pi)
```
### Przykład użycia biblioteki `random`
```python
import random

print(random.randint(0, 100))
```
