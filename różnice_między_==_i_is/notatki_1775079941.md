## Teoria
Różnice między `==` i `is` są kluczowe w programowaniu. `==` jest operatorem porównania, który sprawdza, czy dwie wartości są równe. `is` jest operatorem, który sprawdza, czy dwie zmienne odwożą się do tego samego obiektu w pamięci.

## Przykład z rakietą
Wyobraź sobie, że jesteśmy w kosmosie i mamy dwie rakiety, które mają takie same parametry lotu. Możemy powiedzieć, że są one "równe" w sensie, że mają takie same cechy, ale to nie oznacza, że są to takie same rakiety.

## Zadanie
Napisz program, który sprawdza, czy dwie listy są równe, a następnie sprawdza, czy są one identycznymi referencjami. Użyj `==` i `is`, aby sprawdzić różnice między tymi dwoma operatorami. Przykładowy kod:
```python
a = [1, 2, 3]
b = [1, 2, 3]
print(a == b)  # True
print(a is b)  # False
```
