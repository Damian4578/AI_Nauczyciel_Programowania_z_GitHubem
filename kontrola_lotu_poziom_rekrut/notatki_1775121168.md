## 🧠 Dogłębna Teoria
1. Język Python jest językiem wysokiego poziomu, co oznacza, że jest on bardziej abstrakcyjny i daleki od języka maszynowego.
2. Python używa interpretera do wykonywania kodu, co oznacza, że kod jest wykonywany linia po linii, bez konieczności kompilacji całego programu.
3. Python ma dynamicznie typowany system typów, co oznacza, że typy zmiennych są określane w czasie wykonywania kodu, a nie w czasie kompilacji.
4. Python używa mechanizmów zarządzania pamięcią, takich jak garbage collector, aby automatycznie zarządzać pamięcią i uniknąć wycieków pamięci.
5. Python ma wiele wbudowanych struktur danych, takich jak listy, słowniki i zestawy, które ułatwiają przechowywanie i manipulowanie danymi.

## 🚀 Analiza Systemu Rakietowego
1. Systemy rakietowe są złożonymi systemami, które wymagają precyzyjnego sterowania i kontroli, aby osiągnąć cel.
2. Rakietowe silniki są napędzane przez paliwo i tlen, które są mieszane w odpowiednich proporcjach, aby wytworzyć ciąg.
3. Systemy rakietowe muszą być w stanie radzić sobie z różnymi warunkami, takimi jak turbulencje, wiatr i zmiany temperatury.
4. Systemy rakietowe są często wyposażone w systemy nawigacji, takie jak GPS i żyroskopy, aby pomóc w nawigacji i sterowaniu.
5. Systemy rakietowe muszą być w stanie przetrwać ekstremalne warunki, takie jak wysokie temperatury i ciśnienia, oraz promieniowanie kosmiczne.
