Operatory logiczne są podstawą programowania i są używane do podejmowania decyzji w programie. Są one wykorzystywane do sterowania przepływem programu i wykonywania różnych akcji w zależności od warunków. W tym przykładzie, `x` jest zmienną, która jest sprawdzana przez operator `if`. Jeśli `x` jest większe niż 10, to program wyświetli wiadomość "x jest większe niż 10". Jeśli `x` jest równe 5, to program wyświetli wiadomość "x jest równe 5". W przeciwnym razie, program wyświetli wiadomość "x jest mniejsze niż 5". Rakieta ma system nawigacyjny, który sprawdza, czy jest na orbicie, czy nie. Jeśli rakieta jest na orbicie, to system nawigacyjny wyświetli wiadomość "Rakieta jest na orbicie". Jeśli rakieta nie jest na orbicie, to system nawigacyjny wyświetli wiadomość "Rakieta nie jest na orbicie". Temperatura w rakiecie jest sprawdzana przez program, który wyświetla wiadomość "Temperatura jest za wysoka", jeśli temperatura jest powyżej 100 stopni Celsjusza. Jeśli temperatura jest poniżej -50 stopni Celsjusza, to program wyświetli wiadomość "Temperatura jest za niska". W przeciwnym razie, program wyświetli wiadomość "Temperatura jest w granicach bezpieczeństwa".
