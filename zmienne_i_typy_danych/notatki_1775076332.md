### Definicja zmiennej
```python
a = 5
b = 5.5
c = True
d = "Tekst"
```
#### Typy danych

- `int`: całki naturalne - `5, 10, -5`
- `float`: liczby zmiennoprzecinkowe - `5.5, -5.5`
- `bool`: logiczne - `True, False`
- `str`: napisy - `"Tekst", 'Tekst'`
#### Zmienne na miejscu
```python
a = int(input("Podaj liczbę całkowitą: "))
```
#### Typy danych w trakcie przechowywania
```python
a = 5
type(a)  # Wypisze: <class 'int'>
```
